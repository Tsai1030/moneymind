# MoneyMind

> 一個輕量、極簡、隱私至上的個人記帳 App。
> 資料只存在你的裝置裡 — 沒有註冊、沒有雲端、沒有廣告,離線也能用。

---

## 一眼看懂

- 📱 **手機優先** — PWA 設計,加到主畫面就像 native app
- 🔒 **資料只在你手上** — 全部本機儲存(IndexedDB),沒有後端帳號、沒有雲端同步
- 🧮 **計算機式快速記帳** — 直接打 `120+85+50` 不用先按等號
- 📊 **真正能用的分析** — 不只是漂亮的圖,會告訴你「比上月多 12%」「預估月底會超支」
- 🐱 **會看心情的貓** — 預算用越多,貓的表情越微妙

## 主要功能

### 🏠 首頁 — 看這個月還有多少能花

打開 App 第一眼就是「**本月剩餘可用**」一個大字。

- **剩餘可用** = 本月預算 − 已支出。沒設預算時會引導你去設
- **進度條色階** — 用了 70% 變橙、90% 變紅,讓你不用算數字
- **淨資產 + 今日支出**(可折疊)— 想看的時候展開,平常收起來讓清單佔多一點空間
- **近期紀錄** — 按日期分組,「今日」「昨日」自動標記,點任一筆進明細頁編輯
- **月份切換** — 左上角的月份膠囊,點開選任何過去的月份;切到過去月份時會冒出「返回本月」小按鈕
- **3D 角色** — 右上愛心按鈕切換 3D 模式,有 5 隻可玩的模型(純娛樂)
- **記帳的貓** — 中央那隻 Lottie 貓的姿態會隨預算用量改變

---

### 📊 統計分析 — 把消費結構看穿

回答 5 個核心問題:**花多少 / 花在哪 / 比以前多還少 / 有沒有超支 / 有什麼要注意**。

**頂端固定區(不滾動)**

- **總支出大字卡** — 期間總額 + 一行「↑12% vs 上月同期 · 預估月底 NT$18,000 · 將超支」
- **圖表切換卡**
  - 「圓餅」— 各分類占比,中心顯示**最大支出**(像「餐飲 45%」)
  - 「趨勢」— 本期累積支出折線 + 上期同期虛線疊圖,一眼看出你今天比上個月同一天多花還是少花
  - 右上 chevron 可整張折起來,把版面讓給下方清單

**下方滾動區**

- **子預算進度** — 你在「我」頁面為單一分類設了預算(例如餐飲 NT$8,000),這裡會以進度條呈現,超支的部分顯紅色 overflow
- **分類排行** — 預設 Top 5,可展開全部。環比箭頭只在「差距 ≥15% 且金額 ≥ NT$200」才顯示,避免視覺噪音
- **智能洞察** — 一句白話總結(例:「本月你已支出 NT$X,比上月同期多 12%」)

**「月」和「週」兩種視角**

- 月:跟上月**同期**比(month-to-date,避免「半個月 vs 整個月」假象)
- 週:跟上週同期比

**過去月份查看**

切到 4 月,所有標籤自動變「4月支出」「vs 3月同期」,預估月底自動隱藏(因為 4 月已經結束)。

---

### 💰 資產 — 戶頭餘額 + 投資持倉一次看

頂端「帳戶 | 投資」分頁切換。

**帳戶**

- 支援 4 種類型:現金、銀行、信用卡、電子支付
- 每張卡顯示**計算後餘額**(起始餘額 + 所有收支)
- 新增帳戶時可設定起始餘額
- 刪除帳戶相關紀錄會保留(需重新指定)

**投資**

- 支援 4 個市場:台股、美股、加密貨幣、其他
- 顯示**持倉總市值 + 損益(金額 + %)**
- **自動抓即時報價**(透過 Vercel Edge proxy 打 Yahoo Finance,避開 CORS)
- 台股自動補中文名(從 TWSE / TPEx 官方清單)
- 進頁面自動更新(5 分鐘 cache),也可手動「⟳ 更新報價」強刷

---

### 👤 我 — 預算、外觀、備份

- **本月預算** — 設個總額,首頁進度條才會動起來
- **子預算(分類)** — 點開摺疊區,為每個分類獨立設定預算(例:餐飲 NT$8,000、交通 NT$2,000)。Analytics 的子預算進度條會用這份資料
- **外觀** — 淺色 / 深色一鍵切換
- **備份與還原** — 匯出 JSON、匯入 JSON。換手機時整包搬過去 30 秒搞定

---

### ➕ 快速記帳(任何頁面都能用)

中央 + 號隨時打開記帳輸入面板。

- **計算機輸入** — 支援 `+ − × ÷` 直接算,不用切換到計算機 App。例:聚餐分帳就打 `1200/4`
- **「今天」「昨天」一鍵** — 90% 的時候你只需要這兩個按鈕
- **支出 / 收入** 一鍵切換
- **分類 + 帳戶 + 備註** — 都是必選,但都是一指可達

## 一些刻意的細節

不一定第一眼看得到,但都是有想過的:

- 環比比較用 **month-to-date**,不會拿「半個月」跟「整個月」比,所以數字是公平的
- **預估月底**要月初過了 5 天才會出現,避免「2 號花 500 → 預估月底 NT$15,000」這種噪音預測
- 預測算法是「線性外推 60% + 上月日均加權 40%」,平滑掉前期暴衝
- **環比箭頭有門檻** — 只在差距 ≥15% 且金額 ≥ NT$200 才顯示,免得整列都是箭頭沒重點
- 切到過去月份,標籤跟比較基準會**全部自動跟著換**(本月→5月、上月→4月)
- 鍵盤跳出來不會把整個畫面頂走 — 外殼是鎖住的,只有內層卡片會滾(像 native app)
- 切換「圓餅 / 趨勢」共用同一個 220px 容器,**不會跳版面**

## PWA 安裝(離線、像 App)

裝起來之後就是真的 App,可離線、不吃網路。

- **iPhone (Safari)** — 開網址 → 點下方分享按鈕 → 加入主畫面
- **Android (Chrome)** — 開網址 → 右上 ⋮ → 安裝應用程式
- **Desktop (Chrome/Edge)** — 網址列右側會出現「安裝」圖示

## 隱私

- 所有資料(交易、預算、帳戶、持倉)**都存在你瀏覽器的 IndexedDB**
- **沒有後端帳號、沒有資料庫、沒有分析追蹤**
- 唯一會打外網的場景:**投資頁面抓股價**(走 Vercel Edge proxy 打 Yahoo,只送股票代號不帶任何個資)
- 換裝置 → 用「匯出 / 匯入 JSON」自己搬

## 規劃中

- 🧠 **智能洞察升級** — 用 LLM 解讀消費模式,寫出個人化提醒(例如「週五晚上餐飲是平均的 2.3 倍」)
- 🔁 **重複交易** — 固定月支自動建立(房租、訂閱)
- 📥 **匯入** — CSV / Apple Pay / 信用卡帳單
- 🔎 **分類下鑽** — Analytics 點任一分類 → 看該分類本月所有交易
- 🏷️ **標籤** — 跨分類的橫向 tag(例:「日本旅遊」可以涵蓋多個分類)
